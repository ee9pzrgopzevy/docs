Technical analysis
==================

This documentation covers technical indicators, statistical calculations
and portfolio performance analysis functions.

:py:mod:`tradeexecutor` strategies use :term:`pandas_ta` library for technical analysis calculations.
Currently :term:`pandas_ta` supports over 150 technical indicators.
