About pandas_ta
---------------

:ref:`tradeexecutor` strategies use `pandas_ta` open source library for :term:`technical indicators <technical indicator>`.

`pandas_ta` is a :term:`Pandas` Python library for :ref:`technical analysis`.
Currently `pandas_ta` supports over 150 technical indicator and analysis functions.

`See the pandas_ta homepage for more information <https://github.com/twopirllc/pandas-ta>`__.
